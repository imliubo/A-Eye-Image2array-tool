# A-Eye-Image2array-tool
An image to C array tool for A-Eye board.

![A-Eye](https://github.com/imliubo/A-Eye-Image2array-tool/blob/master/example/A-Eye_example.jpg)

## Usage:

```bash
python A-Eye-Image2array.py A_Eye.jpg
```
